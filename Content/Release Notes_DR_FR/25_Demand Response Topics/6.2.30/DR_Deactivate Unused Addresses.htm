Deactivate Unused Addresses

While working with RouteMatch, many agencies typically build up a large amount of unused addresses that never have trips assigned to them. Having a high number of these unneeded addresses can lead to errors if a user happens to select one of them by mistake. The Addresses module now contains a feature that allows you to bulk deactivate these addresses at the same time.

The system prevents you from removing any addresses that have trips associated to them in the past or future. This is necessary to prevent errors.

How to Deactivate Unused Addresses

  1. In the Addresses module, click the Unused button on the toolbar.

  1. On the screen that appears, enter or select the criteria of the address you would like to remove and then click the Search and Return button. For example, if you would like to deactivate any unused medical addresses, you can search for the "Medical" address type.

  1. Your search results then appear in the main Addresses view. Review the list to make sure these are the addresses you wish to deactivate.

  1. If the addresses listed are correct, click the Bulk Deactivate Unused Addresses button on the toolbar.

  1. Click Yes on the alert that appears to confirm you want to deactivate the addresses.

  1. The system then deactivates the addresses and removes them from the results list.
