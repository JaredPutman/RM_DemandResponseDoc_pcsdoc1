How to Create a New Customer

  1. In the General area, enter a name (title, first, middle, last, suffix) for the new customer (Required). If you are using a nickname for your customer, you can enter their actual name in the Official Name field.
  2. Enter or search for the customer’s home address. Refer to the Addresses section of this guide for further details on selecting an address.
  3. Select a mailing address for the customer. If the mailing address is the same as the customer’s home address, click the appropriate check box.
  4. Enter a primary phone number, a secondary phone number, and an official phone number (with extensions if appropriate).
  5. Enter the customer’s social security number.
  6. Input an Internal ID. The software can automatically create an Internal ID when saving the new customer or you can input an Internal ID before saving the customer. Configuration of this option is available in the settings module under Customer Options. Enter the customer’s birthday. Their age is automatically calculated.
  7. Select whether or not this customer is currently active by clicking the Customer is Currently Active checkbox.
  8. If the customer is elderly, disabled, a child, or low income select the appropriate check boxes.
  9. Select the gender of the customer (male, female, or N/A).
  10.  In the Trip Default Settings area, enter the mobility type of your customer (Required). Define this setting in the customer options in the Settings.
  11.  Select a load and unload time for the customer in minutes. This is how long it takes a customer to enter or exit the vehicle. You have the option of choosing the default for the mobility type you selected (defined in the settings), or inputting an amount of time specific to that customer.
  12.  Enter an associated trip service for the customer.
  13. If one or more attendants accompany the customer, enter the number and mobility types of the attendant(s).
  14. In the Comments section, enter any additional information needed for the creation of the customer. You can enter up to 5,000 characters.
  15. Click Save on the Tool Bar.
