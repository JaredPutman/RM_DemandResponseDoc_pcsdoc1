How to Create a New Favorite Address

  1. Choose a customer and navigate to the Fav Addresses tab.
  2. Click the New button on the toolbar.
  3. In the window that appears, select a category (these are configured in the Settings module).

  1. Search for the appropriate address using the search bar. Refer to the Search for an Address section in the Addresses chapter for details.
  2. Repeat these steps for any remaining favorite addresses for the customer.
  3. Click Save on the Tool Bar.
