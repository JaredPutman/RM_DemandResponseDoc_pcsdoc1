Customers Module Tool Bar

Item Description
New

Click to add a new customer.
Save

Click to save the information for the customer you are currently working on.
Refresh

Click to reload the window.
Customer List

Click to toggle the display of the customer list on or off.
Include

Click to filter customer list between active and inactive customers.

 Inactivate Customers with Expired Eligibility

Click this to inactivate a customer who has expired funding source eligibility. This will delete all trips for the customer from the present date forward.
Customer Trip History

Click to view the history of trips taken by the customer for a selected date range.
Show Customer in Trips

Click to display the current customer in the Trips window.
Delete Customer

Click to delete a customer.
Print

Click to see the available print options for the Customers module.
Search

When searching for customers, you can choose one of the following options:

  • Last Name – Search for the customer’s last name beginning with the text you have entered.
  • Full Name – Search for the customer’s full name containing the text you have entered.
     
  • Confirmation Number – Search for the software-designated confirmation number. Only enter numbers in the search box.
     
  • Customer Eligibility Program ID – Search for the identification number associated with the Eligibility Program recognized by your agency.
  • SSN – Search for the customer’s social security number (formatted or unformatted).
  • Phone # – Search for customer’s phone number (formatted or unformatted).
  • Customer ID – Search for the customer’s internal ID number.
  • Authorization – Enter the authorization code you want to use to search on.
