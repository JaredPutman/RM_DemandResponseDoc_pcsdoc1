How to Add Certification Information

Follow the steps below to add Certification Information to a customer profile.

  1. Choose the customer you want to add certification information for, and navigate to the certification tab.
  2. Enter information for the following dates:
  • Certification Request Date
  • Application Mailed Date
  • Application Received Date
  • Assessment Date
  • Appeal Received Date
  • Appeal Hearing Date
  • Appeal Decision Date
  • Approval Date
  • Renewal Date
  • Renewal Reminder Sent Date
  • Expiration Date
  1. Enter the Letter Code.
  2. Enter the Eligibility Code.
  3. When printing an identification card, be sure to click the check box to indicate it.
  4. Choose a Certification Status from the drop-down list for the customer (Required).
  5. Click Save on the Tool Bar.

In the Questions sub-tab:

  1. Enter answers for the questions displayed on this tab. The Settings module under the Certification Options are where the user can define the Questions and the Assessment Groups. The display order for Certification Questions can also be edited Settings under the Assessment Groups option.
  2. Select dates from the drop-down calendar in the Date From and Date Through fields. These indicate the start date and end date for the certification.
  3. Click Save on the Tool Bar.

In the Custom Fields sub-tab:

  1. The options that display in this tab are set up in the Settings under the Certification Options.
  2. Fill in all the pertinent information.
  3. Click Save on the Tool Bar.

In the Notes sub-tab:

  1. If you need to make any notes for the customer’s certification, click the New button at the top-left corner of the screen.
  2. In the white field that activates when the New button is clicked, enter any needed text.
  3. Click Save on the Tool Bar.
  4. When saving the note, it appears in the grid on the top part of the screen. The software automatically records the creation date of the note, the date when the note was last modified, who created the note, and the last person who modified the note.
  5. If you would like to edit a note, select it and make any needed changes in the lower field.
  6. If you would like to delete a note, select it and click Delete.
  7. The other buttons in the Notes tab allow you to refresh the Notes grid, increase the font size of the notes, and decrease the font size of the notes.
