How to Audit Profile Changes

It is also possible to make notes on the changes by completing the following steps:

  1. Click the plus sign next to Audit Log Summary for Customer to display all changes made to the selected customer.
  2. Select a specific change made to the customer.
  3. Below the main window of the tab, click the New Note icon. The Note field of the tab displays.
  4. Enter text for the note (e.g. describing the nature of the change, the reason for the change. etc.). The blue bar that fills as you type keeps track of the amount of characters in the text in the note. When the blue bar is full, you can no longer enter text in the window.
  5. When all appropriate notes have been entered, click the Save icon.
  6. Notice the saved note underneath its corresponding change. Click the plus sign next to the change to expand the list of all notes related to that change.
