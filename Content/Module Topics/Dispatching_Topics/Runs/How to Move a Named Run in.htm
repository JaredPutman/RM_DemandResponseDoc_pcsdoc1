How to Move a Named Run in Dispatching

You can easily move a named run in the Dispatching window by selecting one or more named runs and moving them to the desired vehicle.

  1. In Dispatching, select the desired run(s) and click the Move Run to Vehicle. The Assign Runs/Breaks to Vehicle window appears.

  1. Select the desired vehicle where you want the named run(s) to reside.
  2. Click Select and Return.

 

When moving a run to another vehicle, the completed trips remain in the original vehicle and the rest of the trips move to the new vehicle with the newly created run. The run name remains the same, but a new run ID is generated.
