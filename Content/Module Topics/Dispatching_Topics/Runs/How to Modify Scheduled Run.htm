How to Modify Scheduled Run Time

From the Run Management screen you can modify the time of a run once it has been assigned to a vehicle.

  1. Click the Run/Driver Management button on the toolbar. The Run/Driver Management screen - Create Run tab displays.
  2. Select the Assign Run/Driver tab.
  3. In the Vehicle and Assigned Runs pane, select a run.
  4. Once the run is highlighted, right-click on the run and select Edit Time from the menu that displays. The Edit Time window appears.

  1. Modify the time, as necessary.
  2. Click the Save button to save the new time and close the window. The changes to the run will be reflected on the Run Management screen.
