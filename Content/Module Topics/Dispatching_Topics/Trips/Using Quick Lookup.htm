Using Quick Lookup

You can search for trips using one of the following types of information:

  • Reservation – The trip’s system-assigned reservation number.
  • Last Name – The customer’s last name. You can enter the entire last name, or search on what the name ‘begins with.’ For example, if searching for Armstrong, you can enter and search on ‘Arms’ or even ‘A’.
  • Customer ID – The unique identification number assigned to the customer by your agency.

Once results are found, the trips that match the search criteria appear in the Trip Records list. If nothing matches, a pop-up message appears to let you know that nothing matches that search.
