How to Validate a Schedule

  1. If you want to validate the dispatching schedule for specific vehicles, select the vehicles in the grid.
  2. Click the arrow next to the Validate Schedule button on the toolbar and select Validate Dispatching.
  3. If the Schedule Validation doesn’t find any items that break the criteria that you have set, the following pop-up displays:

  1. Click OK. The rest of these instructions are not necessary if the schedule is valid.
  2. If Schedule Validation does find items that break the criteria, the Schedule Validation screen displays a list of these items, along with their suggested solutions.

  1. Review the items on the Schedule Validation screen.
  1. If there are more than 25 items total, paging controls appear at the bottom of the screen. The single right-facing arrow pages to the next 25 items. The double right-facing arrows show the last 25 items. The single left-facing arrow pages to the previous 25 items. The double left-facing arrow pages to the first 25 items.

  1. You can filter the list of items to see only the category that you are interested in. You can only filter a list that has more than one category represented. To filter, click the down arrow on the filtering menu in the bottom left corner of the screen and select a category. Select All to view items in all categories.
  1. If necessary, manually solve items:
  1. To have RouteMatch take you to the item, right-click the item on the Schedule Validation window and select Take me to this trip in Scheduling from the pop-up menu. It is easier to see the trip if you do not have the Enhanced Manual Scheduling view enabled.

  1. To return to the Schedule Validation window, click the Validate Schedule button again.
  2. If necessary, have the system automatically solve items. Click the appropriate check box under Here is a way we can automatically solve this for you on the Schedule Validation window.
