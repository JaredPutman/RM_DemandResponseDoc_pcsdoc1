Configuring Quick Lookup

  1. In the Settings module, expand the Dispatching Options.
  2. The following settings are at your disposal for configuring the Quick Lookup:

Setting

Description

Quick Lookup Numeric Entry Defaults to Customer ID

If set to true, Quick Lookup automatically searches for a customer ID number if you enter numerals into the search filed. If false, it automatically searches for the reservation number.

Quick Lookup Show Vehicle Information

Set this to true to display the trip’s real-time vehicle information.

Trip Lookup – ‘GOTO’ Dataview

Click the drop-down arrow to choose one of the data views preconfigured on the Dispatching module to use when clicking Goto on the Quick Lookup dialog or using the shortcut Alt+G.

Trip Lookup – Show User Field

For each of the User fields created on the Trip Options Custom Fields sub form, select True or False. True will show the field on the Quick Lookup, allowing the user to add data. False hides the field from the Quick Lookup dialog. The number is determined by the Field Number chosen on the Custom Fields sub form.
  1. Click the Save icon on the Settings module toolbar. You may need to close RouteMatch and log back in to see these changes on the Quick Lookup dialog.
