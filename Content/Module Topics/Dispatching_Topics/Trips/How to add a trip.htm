How to add a trip

  1. Click the New Trip button to enter a new trip in the Dispatching module. The Add New Trip window appears.

  1. In the Customer Select (Required) field, search for and select or enter a customer from the drop-down list. You can search by Last Name, Full Name, SSN, Phone Number or Customer ID.
  2. For the selected customer, accept the given address or choose a Pickup Address (Required) by searching for it using the Search button. Refer to the section Search for an Address in the Addresses chapter for further information on using the search feature. You may also select the Favorite Trips button to access the Favorite Trips screen to select a trip option. Favorite trips are set up on the Customers window – Favorite Trips tab, but in order to set them up there must be Favorite Trip Categories added by an Administrator in the Settings window – Customer Options.
  3. Choose a Dropoff Address (Required) by searching for it using the Search button. The same search criteria apply here.
  4. Select a Timing Preference (Required). This would indicate whether the trip was a pickup, dropoff, or a pickup with an appointment time.
  5. Enter the Request Time (Required).
  6. The Appt. Time field only becomes active if the Timing Preference of Pickup with an Appointment Time was selected. Enter the Appt. Time, if requested.
  7. In the Schedule Information block, select an option from the Select Vehicle / Run For field. Options include: Day of, Day before, Day after. If the field is set to Day of, you will get a list of vehicles and runs available for the "day of" the trip date. If you set the field to Day before you will get a list of vehicles and runs available on the day before the trip date. Selecting Day after creates a list of vehicles and runs for the day after the trip date.
  8. Choose to schedule the trip to a specific Vehicle Or Run. This list is populated with data for the date indicated by the Select Vehicle / Run For field.
  9. Indicate the Pickup Time and the Dropoff Time in the designated fields.
  10. Select a Funding Source (Required) from the drop-down list. If you need to associate a new funding source for the customer click (ellipsis button) to display the Customer Eligibility window.
  11. Select an Override (Required) from the drop-down list. An override works in conjunction with the funding source when calculating billing and could be, for example, an alternate type of customer pay like cash or a pass card. In Verification, when the RBE calculation is performed, the override is computed to be the actual expected customer pay.
  12. Select Service from the drop-down list.
  13. Select Mobility from the drop-down list.
  14. Enter Attendants from the drop-down list.
  15. Enter Guests from the drop-down list.
  16. Select Trip Purpose from drop-down list.
  17. Select Trip Type from drop-down list.
  18. Select Billing Category from drop-down list.
  19. Select Fare Type from drop-down list.
  20. Enter Trip Comments, if applicable.
  21. Enter Customer Comments, if applicable.
  22. Enter Driver Comments, if applicable.
  23. In the Additional Attributes section of the screen, select Will Call, if applicable.
  24. Select, Service Animal, if applicable.
  25. Select Other, if necessary, and use the Customer Comments field to enter explanation.
  26. Once you have entered data, click one of the following buttons:
  • Save and Continue – to save the data and continue entering information on the Add New Trip window.
  • Save and Return – Save the data. The Add New Trip window closes and the trip appears highlighted on the Dispatching window.
  • Save and Create Return Trip – Save the trip as shown and then create a return trip by swapping the addresses so that the pickup is now the dropoff address and the dropoff is now the pickup address.
  • Cancel and Return – Return to the Dispatching window without saving any of the entered data.

 

Depending on your agency’s operations, you can optimize the day-of trip creation screen to book trips for anonymous customers.  Agencies can use this feature to quickly book taxi-style trips using just the keyboard.  Configure this feature with the New Trip Form options in the Settings module.
