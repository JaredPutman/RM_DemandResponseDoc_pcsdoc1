Keyboard Navigation

Quick Lookup has been designed to allow users to exclusively use keyboard navigation instead of using a mouse. This section describes how to use the feature using only the keyboard.

  1. On the Dispatching module, press Ctrl+L to access the Quick Lookup dialog.
  2. On the Quick Lookup dialog, enter your search in the Reservation/Customer box.
  3. The system predicts your search type based on what you enter, so you may not have to change the search type. If you do need to, use the Tab key to access the search option you need and press the Space bar to select it.
  4. Press Enter to perform the search.
  5. If your search returns multiple customers, use the Up and Down directional keys to navigate the customer names and press Enter to select a name and return to the Quick Lookup dialog.
  6. Review the trips in the Trips for list. If there are multiple trips found, use the Up and Down directional keys to highlight the trip you want to view or edit. As you select a trip, the Real-Time Trip Statistics box is updated with that trip.

 

After your search, pressing Enter closes the dialog. Once you add new data or edit existing data, the Enter key saves the changes to the trip.

  1. To view the trip in a Dispatch Data View, use the shortcut Alt+G (if this has been configured in the Settings module.) See the section Configuring Quick Lookup for information on how to configure this.
  2. With the trip you want to add or edit data for selected, press the Tab key to navigate down through the fields at the bottom of the dialog.
  3. If you need to select an option, use the Space bar to check the box.
  4. If you need to use a drop-down menu, use Tab to select the box and use the Up and Down directional arrows to navigate through the selections. Use Tab to move to the next box.
  5. Once you finish adding or editing the data, press Enter to save the data on the trip.
  6. Press Enter again to close the dialog and return to the Dispatching module.
