How to Create a Bookmark

Complete the following steps to bookmark a map view:

  1. Once you have the map set to the correct location and zoom level, click the arrow next to the Bookmark icon.
  2. Select Add bookmark... from the drop-down menu.

  1. Enter a name for the bookmark in the screen that appears. This name displays in the drop-down list and should accurately describe what the bookmark loads on to the map.
  2. Click the Share this Bookmark checkbox if you would like to make this bookmark available in the other maps found in RouteMatch.  For example, if you share this bookmark in the Scheduling module, you can also load it in the map found in the Addresses module, Dispatching module, etc.
  3. Click the OK button.  The bookmark is now saved in the system.
