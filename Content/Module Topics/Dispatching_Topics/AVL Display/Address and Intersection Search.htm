Address and Intersection Search

The Tools tab allows you to search for addresses and street intersections.  Select these options from the Search drop-down menu.

  • Address Search - If searching for an address, enter the house number, city, and zip code in the corresponding fields.  Then, click the Search button and your results display below.

  • Intersection Search - If searching for an intersection, enter the names of the two streets that make up the intersection.  Then, click the Search button and your results display below.
