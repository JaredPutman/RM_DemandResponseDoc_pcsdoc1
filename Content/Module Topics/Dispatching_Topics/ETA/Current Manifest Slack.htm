Current Manifest Slack

Like the ETA (slack) column, the Current Manifest Slack calculation is the difference between the Stop Time and the ETA (actual) time. Because it is reported at the vehicle level, however, it shows only the slack to the next uncompleted stop (and not for any other stops). Therefore, the same value displays in this column for each row belonging to a vehicle.

This column indicates if the driver is ahead (positive number), or behind (negative number). This column is crucial when creating a "vehicle lateness" data view.
