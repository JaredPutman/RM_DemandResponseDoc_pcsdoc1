How to create the Late Vehicle Data View

  1. From the Data View drop-down menu, select New.
  2. On the Data View Properties dialog, enter a name for the data view. In this example, we use Late Vehicles for the name.
  3. Select Auto save In-place changes of view layout to have the system automatically save the changes you make by dragging and dropping columns. Otherwise, you will have to click Save Grid Layout each time you make changes to the data view.
  4. Select Public View? If you would like to make the view visible to all uses in your agency. If this is not selected, only you will be able to see this view.
  5. Select the data columns you need for your view. See the graphic below to see the columns we are using for this example. Your agency may wish to use additional or fewer columns.
  6. Click Save to save the Data View.
