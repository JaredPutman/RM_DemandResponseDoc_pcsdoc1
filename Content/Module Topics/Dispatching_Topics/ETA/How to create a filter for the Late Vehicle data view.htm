How to create a filter for the Late Vehicle data view

  1. On the Data View Properties dialog, click the Filters/Formatting tab.
  2. Click Add in the Filters section to have the view only the trips that match the criteria you want to show in the new view.
  3. Select Current Manifest Slack in the Field Name drop-down list.
  4. Select Less Than Or Equal To in the Condition drop-down. This is due to the fact that lateness is determined by the amount of slack in the schedule. A negative number indicates more lateness and mathematically, the later number is "less than" a number that is not as late.
  5. Enter an amount in the Value[1] field. This indicates the number of minutes you consider a vehicle late. In our example, we use -21.
  6. Click Save and Return to return to the Data View Properties dialog.
  7. Click Add again to add another condition that will only show trips that are not marked complete.
  8. In the Field Name drop-down list, select Completed.
  9. In the Condition drop-down, select Equals.
  10. In the Value [1] box, enter 0. This means that the trips have not been completed.
  11. Click Save and Return to return to the Data View Properties dialog.
