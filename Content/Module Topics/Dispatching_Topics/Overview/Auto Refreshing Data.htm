Auto-Refreshing Data

Auto-refreshing of data in the Dispatching module is enabled in the Settings module under Dispatching Options. From the Dispatching Options, you can set an auto-refresh alert and also set an audio alert for the start of a countdown to the auto-refresh. If these auto-refresh features are enabled in Settings, then in Dispatching, you will see an auto-refresh counter and hear a beep when the countdown starts.

In this example, the countdown timer (on the toolbar) was set to display within 30 seconds of the auto-refresh of the window. Therefore, 30 seconds before the auto-refresh, the timer displayed red and the seconds began counting down. If enabled, an audible beep will sound when the countdown starts.

From the Dispatching module, you can reset the timer or disable the timer for the current session. Select an option from the Auto-Refresh drop down list on the toolbar. These options are only accessible if the time has been enabled in Settings.

You also can determine how much of your data the system should refresh each time. Limiting this to a specific number of hours (instead of all of your data every times) can help improve the module’s performance.
