How to Group Columns in the Data View

  1. On the Data View toolbar, click the Show Group By Box button. The Group By Box area displays.

  1. Drag-and-drop a column heading from the data view for grouping in the group box.
  2. Drag-and-drop the next column heading for sub grouping of the first selection. To remove a group, drag and drop an item from the Group By Box area, and place it back to the data view.
  3. Repeat steps to group a maximum of four sub groups.
