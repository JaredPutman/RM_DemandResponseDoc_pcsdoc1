How to Edit a Data View's Properties

  1. Click the Data View button on the toolbar, and then select Open from the menu list. The Data View Manager screen appears.
  2. Select a data view from the list, and then click Open and Return.
  3. Click the View Properties button on the Data View toolbar. The Data View Properties screen displays.
  4. Edit any of the data view properties, display fields, filters/formatting, and/or grouping.
  5. Click Save and Return.
