How to Add a Data View Filter

  1. On the toolbar, click the Data View button and select New from the drop-down menu. The Data View Properties screen displays.

  2. Click the Filters/Formatting tab.

  3. Click Add in the Filters section. The Filter Properties screen displays.

  1. Select a Field Name. (Required)

  2. In the Condition list, select a condition type. (Required)

  3. In the Value [1] box, enter data.

  4. In the Value [2] box, enter data, if applicable.

 

The Value [2] field becomes active when you select a condition that requires two values.

  1. Click Save and Return.
