How to send messages

  1. Click the RMLink icon. The RMLink Send Message dialog appears.

  1. Click the Message tab.
  2. In the Message To Send area, select a message from the Canned Messages drop-down menu or enter a message you want to send to the driver in the box. Note the number of characters you are able to use, which your agency can configure in the Settings module (default is 120 characters).

  1. If needed, make any edits to the canned message.
  2. Select the device you want to send the message to from the list of Active Devices.
  3. Select the type of Driver Confirmation you require for the message. This can be either, None (no response necessary), Acknowledge (driver must click OK), or Yes/No (driver has to answer yes or no.)
  4. Select the type of notification you want to send in the Driver Notification area. This can be either a Standard message, or a Popup message.
  5. Select Close window after send? if you want the dialog to close after the message is sent.
  6. Click Send Message.

How to Send a Message to a Vehicle Pool

If needed, you can send an RMLink message to all of the vehicles in a pool. In order to do this, you first need to group the grid by vehicle pool:

  1. In the Send Message screen, click the Group Box button above the grid.

  1. Drag the Vehicle Pool heading into the Group Box. The vehicles are now organized under their respective pools. Click the Save Grid Layout button if you would like to preserve your changes to the grid.

  1. You can then click the checkbox next to a pool to select all of the corresponding vehicles. When you send the message, all of the drivers in those drivers receive the same message.
