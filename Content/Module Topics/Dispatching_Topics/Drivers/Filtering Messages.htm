Filtering Messages

You have the ability to select which messages (by vehicle) will display on the RMLink Messages screen and which messages will be filtered out. All messages will still display when the window is opened. Emergency messages from any vehicle will still popup regardless of the filtering you establish. Follow the steps below to set up message filtering.

  1. Click the RMLink Message Center icon to open the RMLink Messages screen.
  2. Select the Vehicle Filter button to set up filtering. The RMLink Messages screen expands to display as shown below.

  1. Sort the list by vehicle or by pool.
  2. Then, check the checkbox for each of the vehicles/pools for which you want to display messages on the screen. You may click the button to choose all vehicles/pools.
  3. Click the Save Filter button to save your selections.
