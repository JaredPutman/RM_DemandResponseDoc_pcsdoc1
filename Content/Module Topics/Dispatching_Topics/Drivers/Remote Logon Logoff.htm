Remote Logon/Logoff

If a driver should have already logged into RMMobile, or if they should have already logged out, you can remotely do it for them. Do this with the buttons below, which appear in the Dispatching toolbar.

How to log a driver on remotely

  1. In Dispatching, click the Remote LogOn Device button in the toolbar.

  1. Select the device in the list that you want to log on. Use the filter box to search for a specific device, vehicle, or driver.
  2. Click Next.
  3. Select the driver you want to log on to the device.
  4. Click OK log the driver on.

How to log a driver off remotely

  1. In Dispatching, click the Remote LogOff Device button in the toolbar.

  1. Select the device in the list that you want to log off. Use the filter box to search for a specific device, vehicle, or driver.
  2. Click the Logoff icon to log the driver off.
