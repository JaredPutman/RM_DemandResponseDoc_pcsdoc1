Adjust a Driver Schedule

After scheduling a driver, you can adjust the driver’s schedule.

  1. In the Scheduled Trips areas, select a Vehicle.
  2. Click the Schedule driver(s) for the selected vehicle button on the Scheduling window toolbar. The Driver Scheduling window appears.
  3. Select the scheduled driver in the Driver Scheduling area.
  4. Click Driver Scheduling on the toolbar and select the Adjust Scheduled Times option. The Change Time screen appears.

  1. Adjust the Start Time, if applicable.
  2. Adjust the End Time, if applicable.
  3. Click Ok.
  4. Click Save Driver Scheduling and select a save option:
  • Use this schedule for today [mm/dd/yyyy] only
  • Use this schedule for today and all [day of week] in the future
