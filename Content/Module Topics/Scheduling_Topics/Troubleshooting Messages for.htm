Troubleshooting Messages for RouteMatch Scheduling Engine (RSE)

The following table is a list of all messages that RSE reports to the RouteMatch application. These messages may appear in pop-up errors, the application debug log, and in the Unscheduled Trip Statistics message window.

In the RouteMatch Settings module, as part of the RSE Profiles, you can suppress any of these messages by entering the id numbers in a comma separated list. You cannot suppress error messages from appearing in the application, but you can stop specific warnings and messages from appearing in your Unscheduled Trip Statistics message list.
