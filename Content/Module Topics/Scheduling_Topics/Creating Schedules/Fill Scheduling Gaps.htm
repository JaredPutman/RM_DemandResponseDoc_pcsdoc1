Fill Scheduling Gaps

This feature allows you to fill the gaps in a vehicle’s schedule automatically using the RouteMatch Scheduling Engine (RSE). Complete the following steps to use this feature:

  1. You first need to select a garage stop, trip pickup/drop-off, or break from the Scheduled Trips window.
  2. Then, open the Optimize drop-down menu on the toolbar and select Fill Scheduling Gaps. This opens the “Fill Schedule Gaps” window.

  1. The properties of the items you selected in step one appear at the top of this screen for reference. With this in mind, you can address the following configuration options:
  • Schedule Options – This drop-down allows you to turn the following configurations on or off:
  1. The time and distance limiting controls.
  2. Keep the scheduled trips at the same time.
  3. Insert scheduled trips into the gaps.
  4. Insert unscheduled trips into the gaps.
  • Profile – Select the scheduling profile to use to fill in gaps.
  • Max Additional Distance – Use the slider or the number box to designate the maximum number of miles to add to the schedule when the system fills in any gaps.
  • Time Window – Specify a window of time within which the system will try to fill in gaps.
  1. Click the Fill Schedule Gaps button on the toolbar.
  2. The trips available to fill the gaps then display in the bottom half of the screen. Select the trip you would like to add to the schedule and click the Schedule Trip button on the toolbar.
  3. Select an option from the drop-down to schedule the trip to the vehicle.
  4. You can schedule any other trips as necessary, and then close the window when you are done.
