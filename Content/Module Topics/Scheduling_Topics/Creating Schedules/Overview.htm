Overview

There are two methods for creating schedules, and you should use a combination of these methods to create your schedule.

  • Manual Scheduling – The scheduler adds all trips to runs or vehicles manually.
  • RouteMatch Scheduling Engine – The RouteMatch Scheduling Engine (RSE) can recommend vehicles for single trips and can even automatically build entire schedules.

You can schedule trips and breaks to the second by activating the appropriate setting. Refer to the RouteMatch 6 Settings Module User’s Guide for more information.

The following is an overview of the scheduling process:

  1. Select a date from the calendar and a time range (if applicable).
  2. Schedule trips to vehicles.
  3. If your agency uses runs, create the necessary runs, add trips, and then schedule the runs to their vehicles.
  4. Schedule driver breaks.
  5. Adjust times of the scheduled runs, trips, and breaks, if needed.
  6. Commit the schedule, if applicable.
