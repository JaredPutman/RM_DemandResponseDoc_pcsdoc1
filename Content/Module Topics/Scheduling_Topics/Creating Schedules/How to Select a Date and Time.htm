How to Select a Date and Time Range

Select a date and time range to determine which vehicles, trips, and runs to display in the Scheduling module. The vehicles display if their pull out and pull in date/times overlap the selected date and time range. Complete the following steps to view Scheduling data for a specific date and time range:

  1. In the date/time fields at the top of the window, select a calendar date and time span. The default is tomorrow’s date, 12:00 AM, and 11:59 PM.

  1. After you select a date, the window will auto refresh. However, if you select a different time range or the Day Of or Day After options, you must manually refresh the data display. This is indicated by the red box that appears around the Refresh button.

  1. Click the Refresh button to refresh the data displayed in the Scheduling module.
