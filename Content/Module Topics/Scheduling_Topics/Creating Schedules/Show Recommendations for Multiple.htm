Show Recommendations for Multiple Trips

Show Recommendations also allows you to select multiple trips for schedule suggestions by completing the following steps:

  1. Select the trips you want to insert into the schedule.
  2. Click the Optimize button on the toolbar.
  3. Select the Show Recommendations option and the "Trips Selected for Scheduling" screen appears.

  1. Under Schedule Options, select all of the check boxes that apply.
  • Limit Number of Vehicles – allows RSE to consider only the number of vehicles established in Settings.
  • Ignore Services - allows any trip(s) to be scheduled on any vehicle regardless of service restrictions.
  • Schedule outside runs - allows the trip to be scheduled outside of run timeframes.
  • Schedule within runs - allows the trip to be scheduled within run timeframes on either a run or a vehicle.
  • Keep trips at same time – select to keep scheduled trips unchanged when running Show Recommendations.
  1. Click Show Recommendations. The Optimization Results screen appears.

  1. Under View Results, select one of the following options to view any errors or messages pertaining to the trip, if applicable:
  • Summary Statistics
  • Unscheduled Trip Statistics
  1. From the Optimized vehicles list select vehicle(s) by:
  • Click Accept Vehicle to accept an individual vehicle schedule.
  • Click Accept Schedule to accept all vehicle schedules.
  1. Click Apply Schedule.
