IVR Callouts

The IVR Callout function schedules calls to customers based on criteria established in the Scheduling Options section of the Settings module (for more information on these options, please refer to the RouteMatch Admin Guide.) For example, this function would be used to remind customers of upcoming reservations. The IVR Callout function provides a method to queue the messages to be broadcast via the external IVR system and also provides a status report of whether the calls succeeded or failed. This functionality is ONLY for clients with IVR licensing enabled.
