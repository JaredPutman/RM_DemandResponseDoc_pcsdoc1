Batch Update the Denial Status of Trips

In the Scheduled Trips area, you can search for trips meeting several denial status criteria. (These criteria are defined in the Settings window – Trip Options.) Once the trips are identified, you can change the denial status of these scheduled trips. Follow the steps below to batch update scheduled trips in denial status.

  1. From the Scheduled Trips search field, select the search option Denial Status. The search field will display as a drop-down list and the All check box appears to the right of the search field, as shown below. The All checkbox is checked by default and the criteria that displays in the drop down is defined in the Settings module.

  1. Select an option from the drop down list.
  2. Then, click the search button.
  3. If the All checkbox checked, the Scheduled Trips area is searched for all trips that match the selected value and a pop up window displays the search results as shown below. If the All checkbox unchecked, the scheduled tree is searched for the next trip matching the selected Denial Status and that trip is highlighted in the Scheduled Trips area.

  1. From the toolbar, you can change the denial status for all trips that display in the window by selecting an option from the drop down list. Or, you can change each individual trip by clicking the Denial Status field for that trip’s row and then selecting an option from the drop down list.
  2. Click Save and Return to apply the changes and return to the Scheduling window.

 

Locked trips appear with a gray background on the pop up window. Locked trips cannot be updated. Any changes that are saved on the Denial Status pop up window appear in the Trip Audit Log which can be accessed from the Trips window.
