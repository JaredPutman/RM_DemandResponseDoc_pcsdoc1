Validation Rules

The following rules are available to be installed on your database by your RouteMatch Project Manager or Customer Support contact. If none of these rules are installed, this feature only checks for basic database issues and is not operationally helpful to your scheduling team.

Rule Name Description Configurable Parameters

< <value> Revenue Hours: Not enough work on vehicle

This rule finds vehicles with less than the configured threshold amount of revenue hours.

@revenuehours - Minimum revenue hours; Default is 4

Deadhead Trip: Deadhead for this trip is greater than <value> miles

This rule finds deadhead amounts greater than the configured amount of miles.

@deadheadmiles - Maximum deadhead miles; Default is 9

Long Ride (Time): Client is onboard more than <value> minutes

This rule finds trips that longer than the maximum configured duration.

@maxduration - Maximum trip minutes; Default is 90

Extended Trip (Time):  Trip scheduled minutes compared to trip estimated minutes

This rule finds trips where scheduled minutes are greater than estimated minutes greater than the configured amount of time.

@OnboardAddMinutes - Default is 0. Added to product of estimated trip time and onboardfactor to determine the maximum recommended trip time.

@OnboardFactor - Default is 1.5. Multiplied by estimated trip time when calculating maximum recommended trip time in addition to OnboardAddMinutes.

Scheduled Before Window: Trip is before requested time window

This rule finds trips that will be picked up before the requested time window.

None

Slack > <value>m: Time between garage or trip and next trip is greater than <value>

This rule finds deadhead and inline deadhead trips greater than the configured amount of time.

@MaxSlackMinutes - Default is 120.

Trip Miles to Onboard Miles: Low trip miles and high onboard time

This rule finds trips that have low miles compared to high onboard time. For example 1 mile in 10 minutes or 10 miles in 100 minutes.

@MinTripSpeed - Default is 10. This is the minimum number of miles that is acceptable for a rider to travel in 60 minutes.
