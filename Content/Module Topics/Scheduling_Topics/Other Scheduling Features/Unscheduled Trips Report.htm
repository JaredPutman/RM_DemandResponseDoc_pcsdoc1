Unscheduled Trips Report

Print unscheduled trips report by completing the following steps.

  1. In the Unscheduled Trips area, select a way to group the trips from the Group by drop-down list.
  2. Select one of the groups of trips.
  3. Click on the toolbar.
  4. Select an Unscheduled Trip from the list. A preview of the report displays on your screen.
  5. Click in the Preview window to print.
