Searching in the Unscheduled Runs Sub-window

The unscheduled runs sub-window has the following search options:

  • Displayed Text – Standard search for the text entered in the field.
  • Last Name – Search for the customer’s last name.
  • Customer ID – Search for the customer’s internal ID number. This search has to be exact.
  • Confirmation # – Search for the trip’s confirmation number. This search has to be exact.
  • Address – Search for the trip’s pickup or drop-off address.
  • Run Name – Search for the name of a scheduled run.
