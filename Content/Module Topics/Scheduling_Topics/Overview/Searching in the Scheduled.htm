Searching in the Scheduled Trips Sub-window

The scheduled trips sub-window has the following search options:

  • Displayed Text – Standard search for the text entered in the field.
  • Last Name – Search for the customer’s last name.
  • Customer ID – Search for the customer’s internal ID number. This search has to be exact.
  • Confirmation # – Search for the trip’s confirmation number. This search has to be exact.
  • Address – Search for the trip’s pickup or drop-off address.
  • Run Name – Search for the name of a scheduled run.
  • Vehicle Name – Search for the name of one of your agency’s vehicles.
  • Denial Status – Search for the trip’s denial status.

This sub-window also has several search filters. These filters only show the trips that match the criteria entered in the filter. Enter the text to filter on and select a filter type which include the following:

  • Drop-off Address – Filter by the trip’s drop-off address.
  • Pickup Address – Filter by the trip’s pickup address.
  • Drop-off City – Filter by the trip’s drop-off city.
  • Pickup City – Filter by the trip’s pickup city.
  • Drop-off Common Name – Filter by the common name of the trip’s drop-off.
  • Pickup Common Name – Filter by the common name of the trip’s pickup.
  • Trip Purpose – Filter by the trip’s purpose.
