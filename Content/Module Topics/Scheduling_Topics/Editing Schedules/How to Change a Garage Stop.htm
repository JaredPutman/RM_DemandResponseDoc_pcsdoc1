How to Change a Garage Stop Time

  1. Expand the vehicle to display its garage stops.
  2. Right-click other garage stop you want to edit. When using the Enhanced Manual Scheduling sub-window, right-click a vehicle and select Garage Stop Properties.

  1. Select a Time Edit from the menu that appears. You can select to edit the Earliest Pullout/Latest Pullin time, or select to edit the Scheduled Pullout/Pullin time. The Change Garage Time screen appears. The Earliest Pullout time is the earliest time that the vehicle can be scheduled to leave the garage. The Latest Pullin time is the latest time that the vehicle can be scheduled to pull into the garage at the end of the schedule.

  1. Adjust Time.
  2. Select a save option:
  • Save garage for – [mm/dd/yyyy] only]
  • Save garage for this day forward – [day of week] only
  • Save garage for this day forward – [All applicable days]
  1. Click Save And Accept Time.
