How to Change a Trip's Time Acceptance

Once a trip has been scheduled and you have selected a time acceptance option for the trip, you can mark the time as unaccepted by following the steps below:

  1. Select the scheduled Trip for which you want to change the time acceptance option.
  2. Right-click on the trip to display the time acceptance options. The menu displays as follows:

  1. Select a Time Not Accepted option for the trip. Notice that the arrows change to the orange ‘time not accepted’ arrows.
  2. Right-click on the trip and select Time Edit from the acceptance options to adjust the time, or select Accept Time to keep the original time.

  1. If you are changing the time, enter a new time for the trip in the Change Trip Time dialog.
  2. Select Save And Accept Time.
