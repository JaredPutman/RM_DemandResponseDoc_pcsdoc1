Show Trip in Scheduling

You can quickly view a trip in the Scheduling module by completing the following steps. This feature is a helpful time saver and can assist you when searching for a specific trip to schedule.

  1. Navigate to the correct customer and select a trip.
  2. Click the Show Trip in Scheduling button.
  3. The screen switches to the Scheduling module with the trip already selected.
  4. From here you can schedule the trip to a vehicle or run.
