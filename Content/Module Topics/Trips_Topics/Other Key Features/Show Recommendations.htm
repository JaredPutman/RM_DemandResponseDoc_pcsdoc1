Show Recommendations

The Show Recommendations feature found in the Scheduling module can also be accessed from the Trips module. This allows you to schedule the currently selected trip with the assistance of RSE, which is necessary when negotiating trip times (explained above).

  1. Navigate to the correct customer and select a trip.
  1. Click the Show Recommendations button.
  1. This option displays all possible vehicles that can apply to the trip. From this screen, select a vehicle and schedule the trip to it.
  1. For further instructions on scheduling trips, refer to the Scheduling User's Guide.
