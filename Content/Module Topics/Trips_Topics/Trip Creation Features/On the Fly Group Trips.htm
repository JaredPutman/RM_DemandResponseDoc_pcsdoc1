On the Fly Group Trips

The Group Trip Booking feature introduced in 6.2.29 has a new enhancement that allows you to create an impromptu group when booking a new trip. Do this by completing the following steps:

  1. Once you have configured the customer's trip correctly, click the Save button on the toolbar and select the Save and create Group Trips option.

  1. Select the ellipsis (...) option on the screen that appears and then click the Next button at the bottom of the screen.

  1. Select which customers to add to the group. You can use the "control" or "shift" buttons on your keyboard to select multiple customers. Then, click the Add to Group button to move the customers to the right side of the screen and include them in the group.

  1. Click the Next button at the bottom of the screen once you have added all needed customers.
  2. The next screen displays all of the customers you have selected for the group. If this list is correct, click the Next button at the bottom of the screen.

  1. You then need to select a pickup/drop-off option for the trip. You have three options:

  • Use the pickup/drop-off addresses selected for the initial trip
  • Use each group member's default home address as the pickup address
  • Use each group member's default home address as the drop-off address
  1. You also have the option to override the trip's funding source with each group member's default funding source. Do this by clicking the checkbox.
  2. Once the group trip is configured correctly, click the Create Trips button at the bottom of the screen. The system then creates individual trip records for each customer in the group.
