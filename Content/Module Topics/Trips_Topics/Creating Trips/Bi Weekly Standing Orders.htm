Bi-Weekly Standing Orders

You can book bi-weekly standing orders, which create a set of trips for a customer that take place every other week. In order to use this feature, you must activate it in the Settings module by configuring the “Show Standing Order Bi-Weekly Option” (found under Trip Options) to True. You can then complete the following steps to create a bi-weekly standing order:

  1. In the Trips module, select a customer, open a new trip form, configure the trip as needed, and then click the Make this Trip into a Standing Order button.

  1. In the Trip Standing Order pop-up that appears, select which days of the week this trip occurs on and then click the Make Bi-Weekly checkbox.
  2. You can then specify the date range of the standing order. The maximum limit is one year to help save on processing time.
  3. Click the OK button to create the bi-weekly standing order. An alert displays detailing the trips you have now created. Click OK to close the pop-up.
