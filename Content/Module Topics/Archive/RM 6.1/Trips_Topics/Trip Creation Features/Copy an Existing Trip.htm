Copy an Existing Trip

The Copy Existing feature allows you to make a copy of any already-existing trip and apply it to a new date. Again, this functionality can be helpful for general use, but one of the other features explained in this section might be a better option, depending on the situation.

  1. Navigate to the correct customer, select a date on the calendar, and find the trip you wish to copy.
  1. Click the Copy Existing button on the tool bar.
  1. From the drop-down menu, choose whether you want to copy the trip for a single date or multiple dates.
  1. In the screen that appears, select the applicable date or dates.

 

  1. Make any necessary changes to the newly created trip(s). If you do copy the trip to multiple dates, the trips save but you can go back and edit them.
  1. Click Save on the tool bar and select an option from the drop-down.
