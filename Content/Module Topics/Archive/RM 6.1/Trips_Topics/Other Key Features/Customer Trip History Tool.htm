Customer Trip History Tool Bar

The tool bar located at the top of the customer trip history window contains the following features:

  • Customer Search – This field displays which customer’s history you are viewing. You can also use this field to search for a different customer's trip history.
  • Help – Clicking this button displays a pop-up that explains how to switch to a different customer using the Customer Search.
  • Show Trips – Show the selected trip in the Trips module.
  • Show Scheduling – Show the selected trip in the Trips module.
  • Return – Close the Customer Trip History window and return to the previous screen.
  • Selected Date – Can either be a single date, or the beginning of a span of multiple days.
  • End Date – Check this box to activate an end date for the trip history and then choose a date from the drop-down menu.
  • Refresh – Whenever the date is changed, you must click the Refresh button to load the new time span of the trip history.
  • Legend – Click this button to display an explanation of the colors displayed in the window. Rows written in blue have been canceled and those in red are no shows.
  • Default Format – Click this button to reset the organizational tree beneath the tool bar.

Just beneath the Customer Trip History screen's tool bar, it is possible to set up how the information on this screen is displayed. You can build an organizational tree by dragging the column headers to the area just below the tool bar. You can drag multiple column headers into this area to further sort the trip history.
