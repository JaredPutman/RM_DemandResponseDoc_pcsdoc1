How to Create an Unscheduled Run

  1. Click (Run/Driver Management button) on the toolbar. The Run/Driver Management screen displays.
  2. Click the Create Run tab.

  1. Select the Service type of the run from the drop down list.
  2. Enter a Name for the run.
  3. Select the Start Date/Time and End Date/Time the run will take place.
  4. Select the checkbox if you wish to Override Service Default Settings.
  5. Checking the Override box makes the next field active. Choose whether you want the runs to be scheduled using the Optimizer or not.
  6. Select the Save button.
  7. Click the Assign Run/Driver tab to allot the run to a vehicle and/or assign a driver to a run. The tab displays as follows.

The tab has three panes. The top, left pane consists of unassigned runs grouped by Service. The bottom, left pane lists unassigned drivers. The right pane lists vehicles and assigned runs.

  1. In any of the panes, select an option from the drop down date list to display a specified day. Options include: "Day of", "Day after", and "Day before".
  2. To assign a run to a vehicle, select one of the runs in the top, left pane (Unassigned Run) and then select the vehicle to which you want to assign the run in the right pane (Vehicle and Assigned Runs).
    OR
    To assign a driver to a vehicle, select one of the drivers in the bottom, left pane (Driver) and then select the vehicle to which you want to assign the run in the right pane (Vehicle and Assigned Runs).
  3. The Assign button becomes active when items from both panes are selected. Click the button to assign the run/driver to a destination vehicle. The run/driver will be assigned and the screen will close. The data on the Dispatching window will be refreshed and updated.
