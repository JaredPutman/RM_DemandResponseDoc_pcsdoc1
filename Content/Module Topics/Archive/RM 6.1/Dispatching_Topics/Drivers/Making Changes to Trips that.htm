Making Changes to Trips that Affect MDC Status

In Dispatching, if changes are made to trips that overwrite MDC data, when you attempt to save those changes, a message like the one shown below will display. When you display the attempt.

The changes will be applied; however, the MDC device will refuse the changes unless you select from the pop up message. This will update the MDC device with the changes that affect the trip.

 

Before you select confirm that the driver is in the MDC coverage area and be sure that the driver does not make any changes to the device until the operation has completed.

If you click on the pop up message, only the changes that you made to the trip that do not affect the MDC will be applied.

If you select on the pop up message, none of the changes that you made to the trip will be saved.
