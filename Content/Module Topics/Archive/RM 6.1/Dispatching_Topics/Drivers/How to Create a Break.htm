How to Create a Break

Only one day fixed breaks may be created. Flexible breaks cannot be created here, they must be created in the Scheduling module.

  1. In the Dispatching window, select the garage stop for the run containing the break.
  1. Click (Breaks button) on the toolbar. The Break Properties window appears. Note that there is no default break for a run. The Start Time and End Time must be within the run time block. If the Start Time or End Time are out of the run time block, they will appear in red.

  1. Enter Start Time.
  1. Enter End Time. Paris to Stonehenge.
  1. If it is required that the vehicle be empty before the break begins, then select the checkbox next to Require that the vehicle be empty? to indicate it.
  1. Enter an address or search for a Location for the break to take place, if required.
  1. Click Save and Return to schedule the break.
