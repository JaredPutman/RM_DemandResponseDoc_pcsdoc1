How to view messages through RMLink

The RMLink Message screen opens each time a message is received. If you are in the RMLink Send Message screen when the message appears, you will go back to the Send screen after the RMLink Message screen closes.

You may scroll up and down to view messages. If any of the buttons are grayed out, it means there are no more messages in that direction.

  • Messages display with the icon.
  • Commands display with the icon.
