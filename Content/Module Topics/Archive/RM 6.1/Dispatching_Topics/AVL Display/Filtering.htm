Filtering

The Filter tab allows you to filter the vehicles and devices displayed in the Vehicles/Devices tab. Notice that the available options change depending on whether you are in vehicles or devices mode.

You have the following options in vehicles mode:

  1. The Vehicle Usage section allows you to show all vehicles or only show vehicles that have scheduled trips.
  2. The Vehicles section allows you to filter the vehicles based on the following:
  • Vehicle Type
  • Vehicle Name
  • Vehicle Pool
  • Route Name
  • Run Name
  • Driver Name
  1. The GPS section allows you to show vehicles with devices, without devices, or both.

  3. The GPS section allows you to filter the vehicles based on the following:
