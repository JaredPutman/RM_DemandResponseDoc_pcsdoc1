How to Organize Bookmarks

Organize and make changes to your bookmarks by completing the following steps:

  1. Click the arrow next to the Bookmark icon.
  2. Select Organize Bookmarks… from the drop-down menu.

  1. You can complete the following tasks in the Organize Bookmarks screen:
  • Edit - Select a bookmark and click Edit on the toolbar to change the bookmark's name and share status.
  • Default Bookmark - Select a bookmark and click Set as Default on the toolbar.  This bookmark then loads whenever you click the bookmark icon.  The current default is identified by a green circle with a check mark.
  • Delete - Select a bookmark and click Delete to remove the bookmark from the list.
  1. Click OK to save your change
