Vehicle Grid

The Vehicle Grid displays the selected vehicle, listing one row for each of the stops. All of the stops that have already occurred display in black font. The stops that have not occurred yet are grayed out on the grid. When you click on a row in the grid, the stop on the map is highlighted.  The stop rows in the Vehicle Grid are highlighted as the vehicle’s path goes from one stop to the next.

The fields in the Vehicle Grid are defined as follows:

  • Stop# - The stop number assignment.
  • Time - The stop time.
  • Confirmation# - The trip's confirmation number.
  • Customer - The customer's full name.
  • Status - The MDC status of the trip (Completed or Cancelled).
  • Action - A description of what took place at the stop. For example: Pick up @ 456 Stewart Rd.
  • Pickup Auto Arrive - When making a pickup, this is the time when the vehicle was in range of the stop. If this field is empty it means there is no data.
  • Pickup Arrive - When making a pickup, this is the time when the driver clicked the button to indicate he/she arrived at the stop. If this field is empty it means there is no data.
  • Pickup Depart - When making a pickup, this is the time when the driver clicked the button to indicate he/she departed the stop. If this field is empty it means there is no data.
  • Dropoff Auto Arrive - When making a dropoff, this is the time when the vehicle was in range of the stop. If this field is empty it means there is no data.
  • Dropoff Arrive - When making a dropoff, this is the time when the driver clicked the button to indicate he/she was at the stop. If this field is empty it means there is no data.
  • Dropoff Depart - When making a dropoff, this is the time when the driver clicked the button to indicate he/she left the stop. If this field is empty it means there is no data.
