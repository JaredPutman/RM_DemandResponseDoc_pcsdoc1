How to Edit the Data View Properties

  1. On the Dispatching toolbar, click , and then select Open from the menu list. The Data View Manager screen appears.
  1. Select a data view from the list, and then click Open and Return.
  1. Click on the Data View toolbar. The Data View Properties screen displays.
  1. Edit any of the data view properties, display fields, filters/formatting, and/or grouping.
  1. Click Save and Return.
