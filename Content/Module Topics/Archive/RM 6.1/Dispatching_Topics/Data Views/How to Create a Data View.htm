How to Create a Data View

  1. From the tool bar, click the Data View button and select New…
  2. This displays the Data View Properties screen. From here, enter a name for this new data view. If you are creating a public view, be sure that other users can understand what the data view shows, based on its name.

  1. If you would like the system to save in-place changes of the view layout, click the corresponding checkbox.
  2. Select Public View if you want to make the data view visible to all users.
  3. To set this data view to Trip Mode, click the corresponding check box. Leaving the box blank will keep this data view in Stop Mode.
  4. In the Fields area, select the Available Fields you want to display for this data view, and click the right-arrow Add icon. This moves the fields to the Visible Columns list and adds the field to your data view.
  5. When you have all the desired fields selected, choose the order they will display in by using the up and down-facing arrow buttons located above the Visible Columns box, or click and drag the fields into position.
  6. If you mistakenly add a field, you can remove the field by selecting it and clicking the single left-facing arrow button. Click the double left-facing arrow to remove all of the fields.
  7. Click the Save and Return button on the tool bar to close the window and create the data view. If you would like to save and keep working, click the Save button.

 

You can select multiple fields by using the shift or control key on your keyboard.
