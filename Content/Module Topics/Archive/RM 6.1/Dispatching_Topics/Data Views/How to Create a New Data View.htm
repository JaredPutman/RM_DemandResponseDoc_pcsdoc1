How to Create a New Data View from an Existing Data View

  1. On the Dispatching toolbar, click , and then select Open from the menu list. The Data View Manager screen appears.
  1. Select the desired data view from the list, and then click Open and Return.
  1. Click again, and select Save As from the menu. The Data View Properties screen appears.
  1. Make the desired changes to any of the data view properties, display fields, or filters/formatting.
  1. Under the General tab, in the Name field, enter the desired name for the new data view.
  1. Click Save and Return.
