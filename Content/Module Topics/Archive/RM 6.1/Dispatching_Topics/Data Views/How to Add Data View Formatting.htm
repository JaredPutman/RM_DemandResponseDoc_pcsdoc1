How to Add Data View Formatting

A data view may have one or more formatting properties. The fields detailed in the Use Date and Time Fields to Build Filters section of this guide can also be used as formatting properties.

  1. On the toolbar, click the Data View button and select New... from the drop-down list. The Data View Properties screen displays.
  2. Click the Filters/Formatting tab.
  3. Click Add in the Formatting Conditions section. The Formatting Condition Properties screen displays.

  1. Select a Field Name. (Required)
  2. In the Condition list, select a condition type. (Required)
  3. In the Value [1] box, enter data.
  4. In the Value [2] box, enter data, if applicable.

 

The Value [2] field becomes active when a condition that requires two values is selected.

  1. Select Back Color (Required). This allows you to color-code changes. For example, if you set up a formatting condition for the Field Name of Stop Time with the Condition set to Minutes After Now Less Than and the Value set to 30. You could select the Back Color to display Yellow when the Stop Time is less than 30 minutes.
  2. Click the Font Bold? checkbox to make the font bold.
  3. Click the Font Italic? checkbox to italicize the font.
  4. Click the Flash Tab? checkbox if you want to set the tab of your data view to flash. Whenever any row of data matches any condition for which a flash option has been selected, the tab for that view will flash on and off. The flashing will continue until the view has been refreshed and the condition is no longer valid. You can right-click the tab and temporarily stop the flashing; however, the flashing will resume if a refresh occurs and the condition is still valid.
  5. Click Save and Return.
