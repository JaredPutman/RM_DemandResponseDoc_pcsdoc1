How to Reassign a Trip to a Vehicle

This function allows you to reassign runs or trips to a different vehicle. Reassign a trip to a vehicle by completing the following steps.

 

You can also reassign a trip to a new vehicle in the data view by clicking the vehicle in the trip data view and selecting a new vehicle from the drop-down list.

  1. Select a trip (or multiple trips) from the trip list in the Dispatching window.
  2. Click (Reassign Trips button) on the toolbar. The Assign Trips screen appears.

  1. Select a vehicle from the list.
  2. Click Assign Trips. The trip(s) will be reassigned to the vehicle selected.

 

If a red check appears on the vehicle icon, the run has already been verified. Assigning a trip to the run will make the run unverified.
