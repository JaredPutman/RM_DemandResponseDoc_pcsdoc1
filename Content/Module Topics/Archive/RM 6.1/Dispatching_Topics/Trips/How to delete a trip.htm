How to delete a trip

  1. Open the desired data view, enter a Customer Name in the search field, and click the Search button or expand the grouping in the data view to display the customer trips.
  2. Select the trip(s) you want to delete. Multiple trips can be selected and deleted.
  3. Click on the toolbar. A confirmation screen appears.
  4. Select Yes, to delete.

 

Trips chained to the deleted trips will display one by one in a pop-up as shown below. Each pop-up will display the chained trip confirmation number.

  1. Select trip(s) from the pop-up you want to be deleted along with the original trip by checking the checkbox in the Select column next to those trips.
  2. Click Save and Return to complete the action. Selecting Cancel and Return will terminate the action.
