How to Validate a Schedule

    If you want to validate the dispatching schedule for specific vehicles, select the vehicles in the grid.

    Click the arrow next to the Validate Schedule button on the toolbar and select Validate Dispatching.

    If the Schedule Validation doesn’t find any items that break the criteria that you have set, the following pop-up displays:

Click OK. The rest of these instructions are not necessary if the schedule is valid.

    If Schedule Validation finds items that break the criteria, the Schedule Validation screen displays a list of these items, along with their suggested solutions.
