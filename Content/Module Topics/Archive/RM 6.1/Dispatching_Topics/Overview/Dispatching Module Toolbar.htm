Dispatching Module Toolbar

The toolbar contains options which are used to operate the dispatching feature within RouteMatch. The chart below gives information on each of these toolbar options.

Menu icon

Click this button or type Alt + M on your keyboard to access the functions of all of the buttons on the toolbar in one place. This button is enabled in the Settings module under User Properties. Your view may have toolbar only, menu only, or both enabled.

Date

Select the date of the information you want to display o the dispatching grid.

Data View

 

Select from the data view options:

  • New – Create a new data view.
  • Open – Open a saved data view.
  • Close – Close the current data view.
  • Save As – Save the current data view with a new name.

Save

Save changes made to the current data view.

Refresh All Day

Refresh the data view for the selected time frame.

Delete

Click to delete a trip or a break from the current data view.

Validate Dispatching

Validates the current dispatching grid.

Vehicle Filter

Click to activate a vehicle filter or to create or edit a filter.

New Trip

Opens the New Trip window and allows you to add a new trip.. You can choose to schedule the trip to a vehicle or to a run.

Run Assignment

Open the Run/Driver Assignment window to create an unscheduled run and assign the run to a vehicle or assign a driver to a run.

Auto-Refresh

Select to reset or disable (for current session) the auto-refresh audio alert and countdown.

Send Message

Send a text message to drivers with MDC devices in their vehicles. Click on the button to open the RM Link Send Message window.

View Message

Receive messages from MDC devices. The messages display in a message window. Click on the button to view the RMLink Message Center.

Show Date

Show date of trips along with the time of trips in the Details section of the window.

Re-calculate Current Day’s Expected Customer Pay

Re-calculate current day’s expected customer pay.

Total Current Run Customer Pay Expected and Collected

Show total expected customer pay and actual fare box collected for a run, all runs on a vehicle or for all vehicles and runs.

Trip Lookup

Click to launch the Quick Lookup function of the Dispatching module. This function allows users to quickly find trips without having to click around with their mouse.
