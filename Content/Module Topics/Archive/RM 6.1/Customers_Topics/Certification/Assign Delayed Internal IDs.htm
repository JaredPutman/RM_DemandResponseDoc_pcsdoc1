Assign Delayed Internal IDs

When creating a customer, the application automatically sets the Internal ID to the default value defined in the Settings module. If you do not want the software to automatically assign an Internal ID number when you save a new customer, there are two other options you can use:

  1. Delayed Automatic Internal ID Assignment: This option keeps the software from assigning an Internal ID number to a new customer until after saving the customer. When selecting this option, the button appears in the Customers tool bar. Once you save a new customer, you will be able to click the button to have the software assign an Internal ID number to the customer.

When using this option, you must also enter a Delayed Internal Customer Id Default Value. This is what appears in the Internal ID box until assigning a number to the customer.

  1. Manual Internal ID Assignment: This option gives you the power to input Internal ID numbers manually. Notice that now when you create a new customer, the Internal ID box becomes un-shaded. You can now manually input values in this field.
  2. Click Save on the Tool Bar.
