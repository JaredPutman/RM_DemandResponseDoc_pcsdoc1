Printing a Certification Letter

The user can print out certification letters directly from the Customers module. Letters can be set up to generate automatically in Settings under Certification Options. The Letter Templates document also fully explains this process.

To carry out this process, you must enter all of the necessary information, and create the letter templates. When these preliminary steps are accomplished, you can print a certification letter by completing the following steps:

  1. From the Customers module, navigate to your desired customer.
  2. Click the Print icon on the tool bar and select the Print Certification Letter option.

  1. Next, you will need to select a letter template to use. T
  2. The system will then create the letter and display it to you for approval before printing.
  3. Make any needed changes to the letter. Use the tool bar for any regularly used word processing functions.
  4. When you are happy with the contents of the letter, click the Print icon on the tool bar.
