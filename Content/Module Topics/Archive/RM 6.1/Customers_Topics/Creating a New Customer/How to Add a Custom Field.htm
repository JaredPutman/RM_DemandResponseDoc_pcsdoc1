How to Add a Custom Field

If you need to add a custom field follow the steps below.

  1. Go to the Settings window and expand the Customer Options field.
  2. Find the Custom Fields setting within the Vehicle Options field.
  3. Click the ellipsis on the far right side of this setting located next to "<Sub Form>".

  1. In the window that appears, enter a new Field Name, select a Data type for this new field, and choose whether this field is visible in the Vehicles module.
  2. Click Save and Return.
  3. In order to view your newly created Custom fields in the Customer’s module, you will need to refresh the Customers window.
  4. Click Save on the Settings Tool Bar. The new customer creation process is complete.
  5. Click Save on the Tool Bar. The software should prompt you to save anything you have forgotten when you navigate out of the Customers module.
