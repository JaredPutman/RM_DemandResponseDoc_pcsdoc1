Eligibility

  1. Click the Eligibility tab and fill in the following fields for the customer’s funding source eligibility:
     
  • Funding Source – Choose a funding source that applies to the customer (Required). You can create these in the Funding Source Module.
  • Default – Pick a default funding source for the customer—this option is not required. The default funding source will select automatically when creating a new trip. There can only be one default Funding Source.
  • Program ID – An internal identifier for this eligibility.
  • Program Reference – A number or other form of identification that records the program to which the customer is associated.
  • Program County of Record – The county office to which the customer is assigned.
  • Start Date – The effective start date of the eligibility.
  • App Rec Date – The date that the service provider or contracting agency received the application.
  • App Sent Date – The date that the sponsoring transportation agency verified the application for eligibility.
  • App Approve Date – Record the date of approval for the customer’s application for eligibility here.
  • App Reject Date – Record the rejection date of the customer’s application for eligibility here.
  • Exp. Date – The date the eligibility for the application expires.
  • Renew Date – The date when the application will need to be renewed.
  • # Trips Allowed – Number of trips allowed for a period of time which is selected in the Trips Allowed Per column.
  • Trips Allowed Per – The time period that applies to the # Trips Allowed a column. Choices are day, week, month, year or quarter.

Quarters are defined as follows: Quarter 1: Jan 1 – March 31; Quarter 2: April 1 – June 30; Quarter 3: July 1 – Sept 30; Quarter 4: Oct 1 – Dec 31.

  • Comments – Enter any extra information not covered by the other fields.
  • Type – Part of the ADA certification process is defining the type of eligibility. Choose one of the following from the drop-down list: Full, Temporary, or Conditional.
  • Condition – If Conditional is chosen in the Type field, click the ellipsis in this field to add circumstances that apply to this eligibility type.
  • Quotas By Time – Click the ellipsis to designate three-hour timeframes with specified trip limits. If any trip quotas are in place, the day(s) of the week for which a quota is in effect will display in the field. For more information on this, refer to the Manage Trip Quotas section of this user’s guide.
  • Delete Date – Allows the user to set a date to delete the funding source from the customer’s eligibility.
  1. Click Save on the Tool Bar.
