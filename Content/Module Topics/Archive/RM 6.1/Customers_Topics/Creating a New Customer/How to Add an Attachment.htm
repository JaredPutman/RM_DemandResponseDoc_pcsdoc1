How to Add an Attachment

The user can attach documents to a customer’s profile. A customer profile must exist prior to adding any attachments.

  1. Select a customer in the Customers Module.
  2. Click the Attachments icon located next to the Gender box on the General tab. The number displayed in parenthesis correlates with the number of attachments that are already associated with the selected customer’s profile.
  3. Click Attach File.
  4. Navigate to the saved document that you would like to attach. Click Open.
  5. On the Enter Attachment Description dialog, enter a short description for the document. Click Attach.
  6. In the Attachments dialog box, verify that the intended document is present. Click Exit.
