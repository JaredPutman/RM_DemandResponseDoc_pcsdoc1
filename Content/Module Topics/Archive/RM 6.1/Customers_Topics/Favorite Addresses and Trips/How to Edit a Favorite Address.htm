How to Edit a Favorite Address

There are two ways to edit a favorite address in the customer’s module.

  1. If the address’ category has changed, select a new one by clicking the black down-facing arrow, and making a selection from the drop-down menu.

—OR—

If the address itself has changed, search for a new one in the box.

  1. Repeat these steps for any other changes you need to make.
  2. Click Save on the Tool Bar.
