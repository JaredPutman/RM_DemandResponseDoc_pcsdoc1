How to Edit a Favorite Trip

There are two ways to edit a favorite trip in the customer’s module.

  1. If the trip’s category has changed, select a new one from underneath the black down-facing arrow.

—OR—

  1. If the trip itself has changed, click Select to bring up the Previous Trips screen, and choose a new trip.
  2. Repeat these steps for any other changes you need to make.
  3. Click Save on the Tool Bar.
