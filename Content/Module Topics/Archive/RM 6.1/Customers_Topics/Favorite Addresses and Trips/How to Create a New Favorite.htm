How to Create a New Favorite Address

  1. Choose a customer and navigate to the Fav Addresses tab.
  2. Click the black down-facing arrow and choose a category for the address (home, doctor, etc.). These are established in the Settings under Customer Options > Favorite Address Categories.
  3. Search for the appropriate address using the search bar. Refer to the Search for an Address section in the Addresses chapter for details.
  4. Repeat these steps for any remaining favorite addresses for the customer.
  5. Click Save on the Tool Bar.
