How to Add a Customer Abuse Record

You can record customer abuse records on the lower screen of the Suspension tab in the Customer’s module.

  1. Click the customer you wish to add an abuse to and navigate to the Suspension tab.
  2. Click New Item on the bottom half of the screen.
  3. Select a Service Abuse from the drop-down list.
  4. Select a Start Date from the drop down calendar and enter a Start Time for the abuse. 
  5. Enter a Grace Period for the abuse. This is the number of days after the start date that the abuse takes effect.
  6. Select an End Date from the drop-down list and enter an End Time for the abuse.
  7. Enter any additional information in the Comment field.
  8. Changing the abuse points requires a supervisor’s override.  Enter the supervisor’s information in the Overridden By field.
  9. If a supervisor changes the abuse points, the date of the change automatically displays in the Overridden On field.
  10. The amount of points assigned to the abuse is automatically valued at the default number defined in the settings. If you want an abuse to have a different number of points than the default, enter a value in the Points Override field.
  11. Select a Connotation for this abuse.
  12. Select the Object that created this abuse.  At this time, trips are the only supported object.  Click the field’s button to display the Abuse Points Object Search window, select Trip, enter the trip’s confirmation number, enter the trip’s date, and click OK.

  1. The Assigned By field automatically populates with the user currently logged in to RouteMatch.
  2. The Assigned On filed automatically populates with the current date and time.
  3. Click Save and Return on the Tool Bar.

Edit an abuse by selecting the field you wish to edit and make your changes. Be sure to Save your work. Delete an abuse by selecting on the appropriate abuse and click the Delete button.
