Customer Module Tabs

Tabs associated with the Customers window include:

     
  • General – The General tab allows you to view and edit the basic contact and identifying information of the customer.
     
  • Miscellaneous – The Miscellaneous tab allows you to view and edit the customer’s funding source eligibility, emergency contact, assistance needs, medical/disability, and travel restriction information.
     
  • Custom Fields – The Custom Fields tab allows you to enter additional information in custom fields that you create in the Settings window (under Customer Options).
     
  • Audit Log – The Audit Log tab allows you to view the history of changes made to the customer file.
     
  • Favorite Addresses – The Favorite Addresses tab allows you to set up commonly used addresses for the customer. For example, categories like doctor’s office, grocery store, etc.
     
  • Favorite Trips – The Favorite Trips tab allows you to set up trips for the customer that they take on a regular basis. For example, categories like doctor’s office, grocery store, etc.Suspension – The Suspension tab allows you to set up customer suspension of service information based on a predetermined reason.
     
  • Certification – The Certification tab allows you to capture all certification compliance data in conjunction with the approval and continued eligibility determination of applicants.
