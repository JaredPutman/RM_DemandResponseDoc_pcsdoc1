General Information

In the General Information area of the screen, enter identifying details of the billing rule, which includes the following:

Field Name

Definition

Name

The name of this rule, which displays throughout the application. The name should be descriptive so other users can easily identify the rule.

Description

Details that summarize the rule’s intended purpose.

Group

Assign a group to organize the rule in the list on the left side of the screen. Either create a new group by typing its name into the field, or select an existing group from the drop-down list.

Computes

Displays the costs that the billing rule computes. This field is display-only and cannot be edited.

Start Date

Select a start date for this billing rule from the drop-down calendar. The billing rule does not take effect until this date is reached.

Identifier

A descriptive indicator for the billing rule.
