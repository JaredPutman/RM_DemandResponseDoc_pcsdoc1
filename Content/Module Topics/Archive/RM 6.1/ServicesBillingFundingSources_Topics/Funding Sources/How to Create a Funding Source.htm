How to Create a Funding Source

When creating a new funding source, you must enter general information about the funding source and set the billing options for no-shows, attendants, guests, and the applicable contract dates. Do this by completing the following steps:

  1. Select a date from the toolbar. If you need to use funding source immediately, make sure today’s date is selected. Otherwise, this funding source is not accessible until the selected date is reached.
  2. Click the New button. This loads a blank form with the new funding source highlighted at the top of the left pane of the window.

  1. Start by entering a Name for the new funding source (Required).
  2. Address the following optional fields based on your agency’s policies:

Field Name

Definition

Internal Agency ID

Your agency’s internal identification number for the funding source.

Primary Contact

The funding source’s contact information including the first/last name, phone number, and fax number.

Address

The funding source’s office address. Notice that this needs to be a geocoded address.

Contract Information

Relevant information detailing the funding source’s contract. This includes:

  • The contract’s start and end date.
  • Whether a customer’s eligibility program ID is optional, required, or disallowed. A customer’s eligibility program ID is entered in the Miscellaneous tab of the Customers module.
  • Whether or not the funding source bills no-shows, attendants, and guests.

Warning Text

If you would like a warning alert to display when the system is not able to calculate customer pay, click the check box next to "Participate In This Alert." You also need to activate the global default message or enter text for a custom message.

The global default message is entered in the Settings under Trip Options.

ADA Corridor Requirements

Make an ADA corridor required for this funding source by clicking the corresponding check box. If a supervisory override is required for this ADA corridor, click the corresponding check box.

NTD Category Identifier

Select a NTD category identifier. You can define this as an ADA, Sponsored, or Private Pay funding source by clicking the corresponding check boxes.
  1. When you have entered all the necessary information for this funding source, click Save on the toolbar.
