How to Create or Modify an Audit Log Note

You can add notes to audit log entries in the Notes tab of this sub-window. To add a note for an entry:

  1. Select the entry you want to create a note for.
  1. Click the New Note icon to add a new note.
  1. Enter text in the field. You can enter up to 4,096 characters.
  1. Use the other controls to edit the text and the way it appears on the screen.
  1. Click the Save Note icon to save the note to the entry in the audit log.
