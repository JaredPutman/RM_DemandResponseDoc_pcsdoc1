How to Schedule a Paratransit Trip or Run to a Mixed Mode Vehicle

Before you can schedule a paratransit trip or run to a mixed mode vehicle, you must configure the fixed-route run to allow deviations. This is done in RouteMatch Fixed Route Scheduling.

Select Allow deviations on the fixed-route runs you want to be able to schedule paratransit trips with.

Once this is complete, you can schedule the trip using the following instructions.

  1. On the Scheduling Module, select an unscheduled paratransit trip.
  1. Click and drag the trip or run to a mixed-mode vehicle who services a route within the ADA corridor at the time the customer needs to be picked up.
  1. When the pop-up menu appears, select the scheduling option you want to use. RouteMatch determines where in the schedule the customer can be picked up and schedules the trip.
