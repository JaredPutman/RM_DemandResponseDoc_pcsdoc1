How to Validate a Schedule

 

Any updates that another RouteMatch user makes to the schedule while you have the Schedule Validation screen open will not be reflected until you refresh or close the screen and reopen it. For this reason, make sure that no one changes the schedule while you are using Schedule Validation.

  1. Click the arrow next to the Validate Schedule button on the toolbar and select Validate Schedule for All Vehicles or Validate Schedule for Selected Vehicle(s) from the drop-down list.
  1. If Schedule Validation doesn’t find any items that break the criteria that you have set, the following pop-up displays.

Click OK. The rest of these instructions are not necessary if the schedule is valid.

  1. If Schedule Validation finds items that break the criteria, the Schedule Validation screen displays a list of these items, along with their suggested solutions.
