How to Create a Break

  1. Select a date and time from the tool bar.
  1. Select a vehicle from the Scheduled Trips sub-window, and then click the New Break button on the toolbar. The Break Properties screen appears.

  1. The Break Date field is read-only and automatically populates with the date selected in the Scheduling module. This field cannot be edited so in order to change it, you will have to close the window and change the date on the main Scheduling module screen.
  1. Select a start and end time for this break. You can type in these values or use the up and down arrows to change the time.
  1. If this is going to be a flexible break, click the corresponding checkbox and enter a number of minutes for the break’s duration. Making this a flexible break means the driver will take their break at some point during the start and end time, and it will last for the length of time you enter in the Duration field.
  1. If the vehicle needs to be empty (no riders) before the driver can take their break, click the corresponding checkbox.
  1. Select a location for where the driver will take their break. This is the same address search field that is found in other parts of the software.
  1. When you have entered all the necessary information for this break, click Save and Return on the tool bar. There are two options:
  • Save break for – [mm/dd/yyyy only] – Schedule the break for the selected date only.
  • Save break for this day forward – [(Day of the week) only] – Schedule the break for the selected date and for each of the selected days of the week forward. For example, schedule the trip for today and all Wednesdays forward.
  1. The start and end times for the break appear beneath the selected vehicle.
