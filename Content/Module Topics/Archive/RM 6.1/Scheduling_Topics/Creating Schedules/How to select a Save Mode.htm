How to select a Save Mode

  1. In the Scheduling module, click the Save Option button on the toolbar.
  1. Select a save option from the drop-down menu.

  • This Day Only – Runs and trips are scheduled or unscheduled for this one day only.
  • All Weekday Only – Runs and trips are scheduled or unscheduled for this day of the week only.
  • All Applicable Days – Runs and trips are scheduled or unscheduled for all applicable days.
  • Prompt – A prompt appears with the above save options, each time you schedule or unschedule a trip or run. This is the default, legacy behavior.
  1. Schedule or unschedule trips and runs that require this save option.
  1. Change the save option when necessary.
