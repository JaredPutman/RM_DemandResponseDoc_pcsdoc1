How to Create a Run

You can create Runs at any time during the scheduling process. There is no limit to the number of runs an agency can create. Create a run by completing the following steps.

  1. Select a vehicle in the Scheduled Trips area. When viewed using Enhanced Manual Scheduling, click a vehicle in the Vehicle List area.
  1. Click the New Run button on the toolbar and the Run Properties screen appears.

  1. Select a Service. The run can be left unassigned in which case it inherits the service of the vehicle to which it is scheduled.
  1. Enter a Name (Required) for the run.
  1. Enter a Start Time (Required).
  1. Enter an End Time (Required).
  1. The Start Date field is read-only and automatically populates with the date selected in the Scheduling module. This field cannot be edited. You must close the window and change the date on the main Scheduling module screen to change it.
  1. Select the Specify an end date check box, if applicable. Selecting this option activates the End Date field.
  1. Enter an End Date, if applicable.
  1. Select the Override Service default settings check box, if applicable. Selecting this option activates the Runs may be Scheduled using Optimizer field.
  1. Select Runs may be Scheduled using Optimizer, if applicable. Select one of the following optimizer options.
  • Yes – The optimizer will automatically schedule trips to this run.
  • No – The optimizer will not schedule trips to this run. Select this option if you want to control the scheduling of trips to this run.
  1. In the Recurrence area, select recurrent days by clicking the respective days or by clicking on Select All to select all weekdays. Click on the weekday with a check mark to clear it. Click on Select None to clear all weekdays.
  1. Click Save and Return on the toolbar. The run displays in the Unscheduled Runs window for the selected date. You can now schedule trips to this run.
