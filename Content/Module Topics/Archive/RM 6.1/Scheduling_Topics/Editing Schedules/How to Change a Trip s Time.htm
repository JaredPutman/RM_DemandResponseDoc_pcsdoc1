How to Change a Trip's Time

  1. Right-click on the trip pick-up or drop-off you want to edit.
  1. Select Time Edit. The Change Trip Time screen appears. With Enhanced Manual Schedule, you right-click on the whole trip so you must select either Time Edit – Pickup or Time Edit – Drop-off here.

  1. Adjust the trip's time.
  1. Select a save option.
  1. Click Save And Accept Time.
