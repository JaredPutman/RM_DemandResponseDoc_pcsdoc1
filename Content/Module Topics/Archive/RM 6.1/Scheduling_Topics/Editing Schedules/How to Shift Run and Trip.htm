How to Shift Run and Trip Times

A run can be shifted back or forward in time by a selected number of minutes. The associated trip times are shifted as well. The run must be unscheduled to perform this function. Shift a run by completing the following steps.

  1. In the Unscheduled Runs sub-window, select a run.
  1. Click the Show Run Properties Screen button on the toolbar. The Run Properties screen appears.
  1. Click the Shift Run and Trip Time on the toolbar.
  1. Select a shift option:
  • Shift Run for – [mm/dd/yyyy only]
  • Shift Run for This Day Forward – [day of week only]
  • Shift Run for This Day Forward – [All applicable days]

The Shift Run and Trip Times screen appears.

  1. Select one of the options from the Shift Run and Trip Time drop-down menu to determine if you are shifting the run back or forward in time.
  1. Enter the number minutes you want to shift the run buy.
  1. Click OK.
  1. Click Yes on the warning window.

 

Remember to reschedule your run after the times have been shifted. The new times cannot be outside of the vehicle time window nor can the times interfere with other run times on the same vehicle.
