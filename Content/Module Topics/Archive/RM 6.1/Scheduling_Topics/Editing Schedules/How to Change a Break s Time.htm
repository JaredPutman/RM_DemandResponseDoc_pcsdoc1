How to Change a Break's Time

You can change a vehicle break time manually by completing the following steps.

  1. In the Scheduled Trips area, click (plus sign) beside the vehicle to display the vehicle breaks. When viewed using Enhanced Manual Scheduling, click a vehicle in the Enhanced Manual Scheduling area.
  1. Right-click on stop time of the break you want to edit.
  1. Select Time Edit. The Change Break Time screen appears. If viewed using Enhanced Manual Scheduling, select either Time Edit – Start Time or Time Edit – End Time.

  1. Adjust Time.
  1. Select a save option:
  • Save break for – [mm/dd/yyyy] only
  • Save break for this day forward – [day of week] only
  1. Click Save and Accept Time.
