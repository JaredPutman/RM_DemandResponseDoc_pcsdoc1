Tool Bar Controls

The following controls are available on the Scheduling module's toolbar:

Date and Time

Select the date and span of time you will create a schedule for.

Refresh

Click this button to reload the window and update the information contained in the module.

Manual Schedule

Use this tool to schedule trips to runs, trips to vehicles, and runs to vehicles manually.

Unschedule

Use this tool to unscheduled trips from runs, trips from vehicles, and runs from vehicles manually.

Delete/Cancel

Delete or cancel runs and trips.

Freeze/Unfreeze

Freeze a scheduled trip. A frozen trip stays exactly where it has been assigned and cannot be moved to another vehicle. Use the drop-down and select semi-freeze to allow RSE the ability to move the trip only within the scheduled time window.

Move to Vehicle/Run

Move a scheduled trip to a different vehicle. This button is active only when a scheduled trip is selected.

Commit Schedule

Commit a vehicle’s schedule. This button is active only when a vehicle is selected.

IVR

Post callouts to the Interactive Voice Recognition system. You can also use this button to print out the IVR Callout Status Report.

Post to Dispatching and Lock

This icon is only available to customers who are set up for manual posting at installation.

Click this icon to post trips to Dispatching and lock the trips.

New Run

Create a new run.

New Break

Schedule a new break for one of your drivers.

Optimize

Activate the RouteMatch Scheduling Engine and its corresponding features.

Quick Recommendation

Activate the Quick Recommendation tool to assist you in scheduling trips.

Schedule driver(s) for the selected vehicle

Use this tool to schedule drivers using the Driver Scheduling screen.

Customer Trip History

View the history of trips taken by the customer for a selected date range.

Show Trip in Trips Screen

Display the selected trip in the Trips module.

Views

Customize the Scheduling module interface to suit your needs.

Print

Print out various reports associated with the Scheduling module.

Hide or Show the Legend

Click to display or hide the Scheduling module legend.
