Configuring the Interface

Configuring the Scheduling Module's interface involves selecting the sub-windows you wish to see and arranging their position on the screen. You can access the majority of the Scheduling module’s interface options with the Views drop-down on the tool bar. From here, you can show or hide the module’s sub-windows, determine the map’s zoom behavior, and control the display of your agency’s fixed routes on the map. The following list explains each of the View options:

  • Scheduled Trips Window – Select to show or hide the Scheduled Trips sub-window.
  • Enhanced Manual Scheduling Window – Select to show or hide the Enhanced Manual Scheduling sub-window..
  • Unscheduled Runs Window – Select to show or hide the Unscheduled Runs sub-window.
  • Unscheduled Trips Window – Select to show or hide the Unscheduled Trips sub-window.
  • Restore Selected Views – Select to return all the sub-windows to their default positions.
  • Map Window – Select to show or hide the Map sub-window.
  • Audit Log Window – Select to show or hide the Audit Log sub-window.
  • AutoZoom Map – Select this and the map will automatically zoom to the trip or route each time you select one.
  • Smart Map – Select this to allow the map to redraw itself only when necessary. This option will cut down on lag during the scheduling process as the map redraw process usually takes a few seconds to complete.
  • Manual Map – Select this and the map does nothing automatically. Any changes to the map view must be completed by hand.
  • Show Route Lines – Display your agency’s fixed routes on the Map sub-window. The Map sub-window must be active to view the routes and this option is required for printing driver directions.
  • Refresh Route – Select this option to update the route displayed on the map to include any changes.
