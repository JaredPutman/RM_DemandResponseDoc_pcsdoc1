How to Print the Runs by Date Report

From the Verification window, it is possible to print out the Runs by Date report. Complete the following steps to do this:

  1. Select a date from the drop-down list.
  1. Select a run.
     
  1. Click the Print icon on the tool bar. The Report Options dialog appears. Select whether you would like the printed run report to show Verified Runs, Unverified Runs with Trip, Verified Runs Without Trips, or all three.
     
  1. Click the Run button to see a preview of the report. The option to export the report to your computer by clicking the corresponding button is available by clicking the Export Report button from the preview screen.
  1. After reviewing the report, click the Print icon on this screen to print it out.
     
  1. Running this report is useful to ensure that runs have been verified and saved with validation. Click the Group Tree button to display a bar on the side that allows you to browse to a specific vehicle or run.
