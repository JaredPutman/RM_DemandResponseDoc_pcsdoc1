How to Create an Unscheduled Run

  1. Click on the Create Run tab.

  1. Select a service for the run.
  1. Enter a name for the run (required).
  1. Enter a start time for the run (required).
  1. Enter an end time for the run (required).
     
  1. Choose whether or not you want the run to be scheduled using the optimizer. To do this, you may need to override the service’s default settings.
  1. To override the default settings of the selected service, check the appropriate box.
  1. Click Save.
  1. If you need reset the screen and clear all the information you have entered, you can do so by clicking the Clear button.
  1. If you decide to not create a run, click Cancel & Return to close the window.
