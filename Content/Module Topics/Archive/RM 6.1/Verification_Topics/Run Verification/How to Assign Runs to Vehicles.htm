How to Assign Runs to Vehicles

  1. Click the Assign Run/Driver tab.

  1. Use the View button to display the Unassigned Run Pane. Be sure to Refresh the window to include the most up-to-date information.
  1. Find the run you wish to assign by browsing through the Unassigned Run pane or use the run search bar. Click to select the run.
  1. Find and select the correct vehicle from the Vehicle and Assigned Runs pane.
  1. Click the Assign button and choose Assign Run.

 

If the verification analysis has already be run once for the selected day, the Run / Vehicle drop down menu will show any suspect cases that need to be reviewed.
