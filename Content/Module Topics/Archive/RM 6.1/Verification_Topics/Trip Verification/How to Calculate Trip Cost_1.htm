How to Calculate Trip Cost for a Date Range

Calculating cost for all the vehicles of a date range generates the data for all post-scheduling reports. Keep in mind that this process can take several minutes to complete.

  1. Complete the verification process and save the information with validation.
  1. Click the Costs button and select Calculate Date Range. This displays the Pick Date Range screen.

  1. From here, select a start date and an end date by clicking on the ellipsis (…) next to each field.
  1. Click OK. This displays the Cost Calculation Summary screen which contains all the corresponding cost information.
  1. When you are done reviewing the information, click OK and you will need to Save the verification grid again (with or without validation).
