What’s New in This Version

This version of the Verification Module User’s Guide has been updated with the following:

  • Ability to calculate costs on vehicles and trips without first marking the items verified.
  • Removal of the ‘Green V’ run summary button, runs automatically summarized when viewed.
  • Added Print Trip Audit Log Day Of option to Print button.
  • Added verification field Coordination Group to enable coordination for trips in the past.
  • Added new Run Summary Grid.
  • Added map view to Verification.
  • Added Verification module ticketing system.
