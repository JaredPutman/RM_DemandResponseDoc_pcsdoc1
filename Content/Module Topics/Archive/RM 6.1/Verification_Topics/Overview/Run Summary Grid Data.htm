Run Summary Grid Data

The data in the rum summary grid shows summarized statistics for the vehicle that performed the run, and the trips that were completed during the run. For more information on each field, please see the list at the end of the Appendix.

You can click the Edit the current data view icon to select or remove fields from the Run Summary Grid. You can also use it to set a color scheme for data that appears in the grid.

For example, you may want unverified runs to appear red in the grid, and verified runs green. For more information about how to create Data Views, please refer to the Dispatching User’s Guide.
