Choosing Run Mode or Vehicle Mode

To use Verification in Run Mode or Vehicle Mode, navigate to the Settings module and expand the Verification Options. Select the drop-box next to the Run Mode setting and choose either true or false:

  • True – This option means you must select a run first and then a vehicle associated with that run when performing verification. A run can have multiple vehicles associated with it.

  • False – This option means you must select a vehicle first and then a run associated with that vehicle when performing verification. A vehicle can have multiple runs associated with it.

Be sure to save after selecting an option. When you make this change, you will probably have to close and restart RouteMatch before the change will apply.
