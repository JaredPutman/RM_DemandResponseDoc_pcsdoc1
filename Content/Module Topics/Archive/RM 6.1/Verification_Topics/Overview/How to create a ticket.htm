How to create a ticket

  1. Select the disputed trip or run, click the Dispute button on the toolbar, and then select Dispute Trip or Dispute Run.

  1. The disputed control displays, allowing you to enter a summary, relevant details, a category, the priority, and a work queue. The categories displayed are determined on the type of ticket created. The trip type displays by trip categories and any other relevant data.

  1. Click the Create button.
  2. In the Ticket control grid, tickets display by group or priority the list of open tickets on the given date.

  1. Selecting a specific ticket allows you to view details of the ticket.
  2. Double-clicking a ticket opens the Ticket Review tab, which allows you to view the ticket’s information and make any updates. This tab is also where you resolve tickets once their corresponding issues have been taken care of.

 

To enable this feature, select True in the Settings module in Verification Options for Activate Ticket View Window.
