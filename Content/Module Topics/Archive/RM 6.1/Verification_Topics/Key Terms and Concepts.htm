Key Terms and Concepts

Term or Concept

Definition

 Validate

Approve information to be included in reports.

 Run Summary

The information about the selected run displayed in the top section of the Verification screen.

 Trip Details

The fields that display all the information pertaining to the trips contained in the selected run. These fields must be validated in order to create reports.

  Verification Grid

The grid made up of all the trip detail fields. This grid takes up the majority of space in the Verification window.
