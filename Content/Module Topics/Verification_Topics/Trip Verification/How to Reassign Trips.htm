How to Reassign Trips

If you need to place a trip on a different vehicle or run, complete the following steps:

  1. Select a trip (or use Shift to select multiple trips) and click on the Reassign Trips button.
  2. This brings up the Assign Trips screen. From here, set the date as day of, day before, or day after.
  3. Perform a search for the vehicle or run you wish to assign the trip to. You can also browse the vehicles/runs by expanding the vehicle pool fields. When doing this, you do have the option to expand or collapse all the fields at once.
  4. Select a vehicle or an assigned run and click Assign Trips.
