Breaks

If your driver took a break during their shift and it was not assigned in Scheduling or Dispatching, you can factor it into Verification by completing the following steps:

  1. Have a run selected in the Verification window. 
  2. Click the Breaks button, this brings up the Break Verification screen.

  1. Click the New button.
  2. Enter a start time and an end time for the break.
  3. Enter odometer readings for the beginning and end of the break.
  4. Select what type of break this was from the drop-down list. Break types are defined in the Settings under Verification Options.
  5. When you have entered all of the information correctly, click Save and Return. Doing so switches the Verified column from false to true.
