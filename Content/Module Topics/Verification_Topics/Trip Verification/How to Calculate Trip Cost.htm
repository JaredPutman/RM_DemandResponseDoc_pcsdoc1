How to Calculate Trip Cost for an Individual Vehicle

Calculating cost for an individual vehicle does not generate data for post-scheduling reports, it only displays for you to review. You can calculate for verified or unverified trips.

  1. Click the Costs button and select the appropriate option, Calculate Current Vehicle or Calculate Current Vehicle Without Mark Verified. This displays the Cost Calculation Summary screen which contains all the corresponding cost information.
  2. When you are done reviewing the information, click OK and you will need to Save again.
