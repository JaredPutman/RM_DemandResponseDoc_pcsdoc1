How to Create an Unscheduled Run

  1. Click on the Create Run tab.

  1. Select a service for the run.
  2. Enter a name for the run (required).
  3. Enter a start time for the run (required).
  4. Enter an end time for the run (required).
  5. Choose whether or not you want the run to be scheduled using the optimizer. To do this, you may need to override the service’s default settings.
  6. To override the default settings of the selected service, check the appropriate box.
  7. Click Save.
