Run Summary Grid Fields

The following is a list of the fields available to view in the Run Summary Grid.

Field

Description

Verified

A checkbox to show if the run is verified or not.

Vehicle Name

The name of the vehicle that performed the run.

Run Name

The name of the run being summarized.

Driver Name

The name of the driver that performed the run.

Actual Pull Out Time

The actual time the driver pulled out of the garage. This data is either entered manually in verification, or populated by RMMobile.

Actual Pull In Time

The time the driver pulled into the garage at the end of the run. This data is either entered manually in verification, or populated by RMMobile.

Trip Count

The number of trips that made up the run.

Passenger Count

The total number of passengers that rode during the run.

Actual Service Hours

The amount of time the vehicle was in service during the run.

Scheduled Service Hours

The amount of time the vehicle was scheduled to be in service during the run.

Actual Revenue Hours

The amount of time during the run that can be considered revenue hours.

Scheduled Revenue Hours

The scheduled amount of time during the run that can be considered revenue hours.

Scheduled Service Miles

The distance the vehicle was scheduled to drive during the run.

Actual Service Miles

The distance the vehicle actually drove during the run.

Scheduled Revenue Miles

The miles that the vehicle drove while in service that are considered revenue miles. These are the miles that can be billed to a funding source.

Actual Revenue Miles

The actual miles that the vehicle drove while in service that are considered revenue miles.

Actual Dead Head Miles

The number of miles that the vehicle was in service that are not considered revenue miles.

Actual Dead Head Hours

The amount of time that the vehicle was in service that are not considered revenue miles.

Exceptions

The total number of exceptions identified on the trips in the run. This is created when the trips are analyzed.
