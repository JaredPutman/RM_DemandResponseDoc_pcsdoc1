How to Move a Run to a Vehicle

If you need to assign the selected run to a vehicle, complete the following steps:

  1. Have a run selected in the Verification window.
  2. Click the Move Run to Vehicle button which brings up the Assign Runs/Breaks To Vehicle screen. From here, set the date as day of, day before, or day after.
  3. Perform a search for the vehicle you wish to assign the run to. You can also browse the vehicles by expanding the vehicle pool fields. When doing this, you do have the option to expand or collapse all the fields at once.
  4. Select a vehicle and click the Assign Runs/Breaks button to assign the run to the selected vehicle.
