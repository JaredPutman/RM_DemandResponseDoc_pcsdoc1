Data Display

The information displayed in the Verification module can be customized by each user. The following is an explanation of how to customize the Verification module to suit your needs. There are three main options that you need to set in order to do this:

  1. Choose whether you want to view the information by run or vehicle.
  2. Decide to view the trips in Stop Mode or Trip Mode.
  3. Select which trip detail fields to display.
