Verification Module Search Options

The search bar is located on the Verification module tool bar. There are several ways to search for information in this module:

  • Last Name – Search using the customer’s last name beginning with the text you have entered.
  • Full Name (Last, First) – Search using the customer’s full name containing the text you have entered.
  • Confirmation Number – Search using the software-designated confirmation number for the trip. Only numbers may be entered in the text box.
  • SSN – Search using the customer’s social security number (formatted or unformatted).
  • Phone # – Search using the customer’s phone number (formatted or unformatted).
  • Customer ID – Search using the customer’s internal ID number.
  • Address – Search using the stop address of a trip. This information is stored in the Stop Address field of the Verification grid.
  • Eligibility Program ID – Search using the customer’s Eligibility Program ID.
  • Trip Authorization – Search using the trip authorization ID that was previously supplied to the customer.
  • Search Grid – Searches using the entered text in all the visible fields of the current Verification grid.
