Selecting Stop Mode or Trip Mode

 The Stop Mode button located on the tool bar to use Verification allows you to toggle between viewing trips by either Stop Mode or Trip Mode:

  • Stop Mode – When the Stop Mode button is selected, pickup and drop-offs for each trip are listed on two separate rows. Pickups are indicated by a P and drop-offs are indicated by a D.

  • Trip Mode – When the Stop Mode button is not selected, trips are listed in Trip Mode. Each trip is listed on a single line with pickup and drop-off times in separate columns. The columns are titled PU Time (pickup time) and DO Time (drop-off time).
