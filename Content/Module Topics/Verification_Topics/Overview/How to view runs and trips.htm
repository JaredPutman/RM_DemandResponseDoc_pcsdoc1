How to view runs and trips

  1. Select the Vehicle Pool that contains the vehicle that performed the run.
  2. The Run Summary Grid displays all of the runs that were performed on the Active Date.
  3. If you want to further filter the list, click the Run Search drop-down to select a specific run in the selected vehicle pool. The selected run appears highlighted in the grid.
  4. You can also limit the list to those runs that match the Verified Status filter. This lets you show all runs, only those that are verified, those not verified, and those that have trips with exceptions.
  5. Select the run you want to view. Trips appear in the verification grid below.
