Selecting Trip Detail Fields

In Verification, you have the ability to choose which trip detail fields to display in the Verification grid. To configure these fields, complete the following steps:

  1. Click on the Grid Settings button, this brings up the Verification Fields window.

  1. Scroll through the list and click the checkbox in the visible columns for each field you wish to include in the grid. In order to keep the Verification process as simple as possible, only select those fields that are absolutely necessary.
  2. Choose the order of the fields by entering a number in the Order column or by using the Move Up and Move Down buttons located on the tool bar. To move a field to the very top or very bottom of the visible columns list, select the Move Top or Move Bottom button.
  3. Choose whether these fields will become your system-wide default settings or just your own personal settings.
  4. When you have made your selections click Save.
  5. If you would like to tweak the order of columns further, click and drag the column headers of the Verification grid into new positions.
  6.  If the Auto Save In-place of Verification Layout button is not selected, you can click the Save Grid Settings button to fully save all the changes you have made.
