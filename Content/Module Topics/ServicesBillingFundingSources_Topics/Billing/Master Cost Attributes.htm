Master Cost Attributes

The Master Cost Attributes tool allows you to set up a billing rule that includes the entire set of all your agency’s billing rules. This can be helpful when transferring billing rules during a software upgrade.

  1. You first need to create a billing rule that computes cost with the Master Trip Cost Function option.
  2. Click the Master Cost Attributes button on the toolbar to display the Master Cost Billing Rule Attributes screen.
  3. From this screen, select an RBE rule from the drop-down list. This list is made up of all the rules that have Master Trip Cost Function selected for how they compute cost.
  4. When you make a selection, the left side of the screen populates with a list of all the billing rules that have been created for your agency.
  5. This list is organized by: Funding Source->Billing Rule->Parameters in a collapsible list.
  6. Click a parameter from the list on the left and its details display in the box on the right.
  7. In the box, you can define a new value for this parameter, edit the existing value, or delete the existing value. If all the parameters are removed, their corresponding billing rule and funding source names turn red.
  8. Changes made to this window are saved automatically.
