Expected Customer Pay

These fields determine how much a customer is expected to pay when they take a trip under this billing rule.

Field Name

Definition

Amount

The specific amount you expect the customer to pay when they take a trip. This value displays on the driver’s manifest (digital or paper) so they know how much money to collect from the customer.

Set How Expected Customer Pay Computed

Select a method for computing the expected customer pay amount. This information displays in the Trips module:

Use Current Billing Rule Group – The expected customer pay is calculated using the name of the billing rule’s group.

Use Current Billing Rule Name – The expected customer pay is calculated using the name of the billing rule.

Use Trip’s Funding Source and Override – The expected customer pay is calculated using the names of the trip’s funding source and override.
